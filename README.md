# 末光咏叹

> 一个纯文字、高自由度的放置挂机爬塔游戏。无需操作，策略至上。

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

**一段关于光与暗的闲置冒险，构筑你的专属技能轴，征服无尽的虚空领域。**

---

## ✨ 核心特色

- **🧠 策略构筑，拒绝无脑**：独创技能轴系统，起手爆发、GCD循环、oGCD瞬发穿插，由你亲手编排战斗逻辑。
- **⚔️ 无限挂机，轻松护肝**：战斗全自动执行，离线收益不间断，哪怕只看一眼也能战力飙升。
- **🎒 无限背包，告别清理**：彻底移除背包容量限制，所有战利品照单全收，再也不用纠结留谁弃谁。
- **⚗️ 终焉精炼，极致追求**：收集终焉装备提取精华，递减式精炼打磨属性直至理论极限，追求完美强迫症的福音。
- **💎 宝珠镶嵌，自由定制**：三孔自由镶嵌，跨越装备基础限制，打造独一无二的神兵利器。
- **💾 本地存档，数据无忧**：所有数据本地持久化，支持一键导出/导入文件备份，换设备也能无缝继承。

---

## 🚀 快速开始

本项目为纯前端应用，无需任何后端服务器或数据库支持。

1. **克隆仓库**：
 ```bash
 git clone https://github.com/xialongxl/Idle_Game.git
 ```
2. **打开游戏**：
 由于项目使用了 ES6 模块化 (`import/export`)，直接双击 `index.html` 打开会因浏览器安全策略报错。
 - **推荐做法**：使用 VS Code 安装 `Live Server` 插件，右键 `index.html` 选择 "Open with Live Server"。
 - **极客做法**：在项目根目录运行 `python -m http.server 8080`，随后在浏览器访问 `http://localhost:8080`。

## 🎮 玩法介绍

### 战斗与技能轴
怪物会不断袭来，你的角色将**全自动**执行你预设的技能序列进行战斗。
- **起手期**：战斗开始时优先释放设定为"起手"的技能，打出首发爆发。
- **循环期**：起手技能进入冷却后，按顺序循环释放常规技能。
- **瞬发穿插**：oGCD（瞬发）技能不占用公共冷却，只要冷却完毕将无视顺序自动瞬间穿插释放！
- **序列分享**：支持将技能轴导出为 `.txt` 文件，或导入他人的作业文件（`@` 标记起手，`#` 撰写注释）。

### 装备与成长
击败敌人掉落的装备是变强的核心途径，越高级的装备词缀越丰富。
- **无限背包**：拾取无上限，告别选择困难。
- **双锁保护**：常规🔒锁定防误分解；📌防换锁防止被高评分新装自动替换。
- **强化突破**：消耗金币强化装备，基础属性指数飞跃，最高可达 +12 阶。
- **宝珠镶嵌**：每件装备三个孔位，镶嵌宝珠跨越装备词缀上限提供百分比加成（同类宝珠全身最多生效3个）。
- **终焉精炼**：分解最高品质的【终焉】装备获得"终焉精华"。消耗精华可对终焉装备的副属性进行精炼，提升值按次数递减，逼近该部位理论天花板。

### 营地与策略
在星盘营地中休养生息，调整状态迎接深层挑战：
- **自动熔炼**：下拉框设定阈值，低于该品质的掉落装备自动分解为金币，终焉装备还能额外产出精华。
- **安全血线**：设定最低生命百分比，战斗中血量低于此线将自动触发紧急撤退，保命防暴毙。
- **瞬间痊愈**：花费 50 金币瞬间回满生命与法力，推图续航的保障。

### 存档与备份
点击左侧面板的"📁 导出存档"按钮，可将当前进度打包为 `.json` 文件下载保存。换电脑或重装系统前，通过"📂 导入存档"读取文件即可完美继承进度。

## 🛠️ 开发者与 GM 指令

游戏内置了 ARCHITECT 终端（GM面板），用于快速调试体验。
- 默认隐藏，未开启不影响正常游玩。
- 偏爱钻源码的开发者，可以尝试在浏览器控制台中输入 `window.enableGM('密码')` 来尝试解锁它。密码就藏在源码的某个角落里 😉

## 📂 项目结构

- `index.html`：游戏唯一的页面入口与 UI 骨架。
- `js/`
 - `main.js`：应用入口，负责实例化核心模块、绑定全局事件及存档导入导出逻辑。
 - `engine.js`：游戏核心引警，包含战斗驱动循环、实体模型、装备生成与词条计算逻辑。
 - `ui.js`：UI 控制器，负责所有面板渲染、背包交互、精炼逻辑及技能序列解析。
 - `data.js`：纯数据层，定义技能、装备品质、槽位、宝珠等基础配置表。
