[末光咏叹 / Idle RPG]

一个纯文字、高自由度的放置挂机爬塔游戏。无需操作，策略至上。

✨ 核心特色

· 🎮 纯文字挂机：没有任何图片或动画，只有纯粹的数字和策略。战斗实时推进，支持1倍至10倍速调节。
· 🧠 深度技能序列：14个技能槽位自由编排，支持 GCD / oGCD 机制。你可以自定义起手爆发和常规循环，实现无缝技能链。
· ⚔️ 高自由度装备系统：装备动态生成，拥有“破损”至“终焉”九大品质。独特的“武器+攻击/防具+防御”隔离机制，告别词条污染。
· 💎 宝珠镶嵌与Build构建：通过镶嵌不同宝珠，自由调整攻击、生命、暴击、冷却等属性，打造属于自己的流派（速攻、暴击、铁桶等）。
· ⛺ 营地与风险机制：每5层设安全区，战败退回。自由设置“自动熔炼”阈值与“安全血线”，在推进效率和生存之间寻找平衡。

🕹️ 快速开始

1. 克隆仓库
   ```bash
   git clone https://github.com/xialongxl/Idle_Game.git
   ```
2. 打开游戏
   直接双击 index.html 文件，或在 VS Code 中使用 Live Server 插件打开。无需安装任何依赖。
3. 开始爬塔
   点击“探索出发”即可开始挂机。你可以在右侧面板编排技能序列。

🗺️ 游戏核心机制

战斗与技能

· GCD 与 oGCD：大部分技能受公共冷却（GCD）制约，而 oGCD 技能可无视 GCD 瞬间释放。合理的编排能让伤害最大化。
· 优先级循环：战斗会严格按照你设定的技能顺序尝试释放。若技能在CD或蓝耗不足，会自动尝试下一个。
· 持续效果：支持 DoT（持续伤害）、HoT（持续治疗）、护盾、易伤、Buff等多种效果叠加（易伤为乘算）。

装备与成长

· 动态生成：所有装备均由算法生成，楼层越高，高品质爆率越高。
· 词条隔离：武器只带攻击词条，防具只带防御词条，首饰可带攻防及稀有副属性（暴击、冷却、共鸣）。
· 强化与宝珠：装备可消耗金币强化至 +12；通过击败Boss获取宝珠，镶嵌后提升对应属性。

营地与策略

· 安全层：每5层为安全区，战败或手动撤退将回到上一个安全区。
· 自动熔炼：可设置品质阈值，低于阈值的装备将被自动分解为金币，让背包保持清爽。

⌨️ 开发者与GM指令

项目原生支持一些调试指令（在浏览器控制台 F12 中使用）：

· gm('密码')：解锁GM面板。
· gm(1) / gm(2) 等：执行对应GM功能（升级、加钱、无敌等）。
· 具体密码和功能请参考源码 main.js 中的 GMTools 实现。
