## 核心特色

- **纯文字挂机**：没有任何图片或动画，只有纯粹的数字和策略。战斗实时推进，支持 1 倍至 10 倍速自由调节。
- **深度技能序列**：最多 15 个技能槽位自由编排，支持 GCD（公共冷却）与 oGCD（独立冷却）机制。可自定义起手爆发与常规循环，实现无缝技能链。
- **高自由度装备系统**：装备动态生成，拥有从“破损”至“终焉”的九大品质。独创“武器+攻击/防具+防御”词条隔离机制，告别词条污染。
- **宝珠镶嵌与流派构建**：通过镶嵌生命、攻击、暴击、共鸣、终焉回响等宝珠，自由调整属性，打造速攻、暴击、铁桶或终焉爆发等流派。
- **营地与风险机制**：每 5 层设立安全区，战败退回。可自由设置“自动熔炼”品质阈值与“安全血线”，在推进效率和生存之间寻找完美平衡。

## 快速开始

1. **克隆仓库**
   ```bash
   git clone https://github.com/xialongxl/Idle_Game.git
   ```
2. **打开游戏**
   直接双击 `index.html` 文件，或在 VS Code 中使用 Live Server 插件打开。无需安装任何依赖。
3. **开始爬塔**
   点击界面上的“探索出发”按钮即可开始挂机。你可以在右侧面板自由编排你的技能序列。

## 游戏系统详解

### 战斗与技能

- **GCD 与 oGCD**：大部分技能受公共冷却（GCD）制约，释放后触发一段全局冷却时间；而 oGCD 技能可无视 GCD 瞬间释放。合理的编排能让伤害最大化。
- **优先级循环**：战斗会严格按照你设定的技能顺序尝试释放。若技能在冷却或法力不足，会自动尝试下一个，确保输出不间断。
- **起手爆发机制**：可将部分技能设为“起手”，系统在每场战斗开始时优先尝试释放这些技能，适合护盾、HoT、易伤等前戏技能。
- **持续效果**：支持 DoT（持续伤害）、HoT（持续治疗）、护盾、易伤、增益 Buff 等多种效果叠加。其中易伤效果为乘算叠加，多个易伤同时存在时收益更高。

### 装备系统

- **动态生成**：所有装备均由算法实时生成，楼层越高，高品质装备的爆率越高。1000 层后终焉装备爆率达到上限。
- **九大品质**：破损、普通、精良、卓越、史诗、传说、神话、圣物、终焉。品质越高，主属性倍率和副属性范围越强。
- **词条隔离**：武器只携带攻击类词条，防具（头、身、腿）只携带防御类词条，首饰（戒坠、遗物）可携带攻防及稀有副属性（暴击、冷却缩减、共鸣）。
- **强化系统**：装备可消耗金币强化至 `+12` 阶，每次强化提升主属性和副属性数值。
- **自动熔炼**：可在营地设置品质阈值，低于阈值的装备在掉落时自动分解为金币，保持背包清爽。
- **背包无上限**：背包容量已移除，无需担心背包满的问题。

### 宝珠镶嵌

- 每件装备拥有 3 个宝珠孔位，可镶嵌不同宝珠。
- 同类宝珠全局最多生效 3 个，超出部分无效。
- 现有宝珠类型：
  - 生命宝珠：最大生命值 +45%
  - 攻击宝珠：攻击力 +45%
  - 共鸣宝珠：共鸣（全能） +30%
  - 暴击宝珠：暴击率 +20%
  - 终焉回响：终焉技能冷却时间缩减 20%

### 营地与风险机制

- **安全层**：每 5 层为一个安全区。在安全区可等待技能冷却、回复状态、调整装备和序列。
- **战败惩罚**：战败或手动撤退将退回至上一个安全区，不掉落任何物品。
- **自动撤退**：可设置安全血线百分比，当角色血量低于该值时自动撤回营地，防止暴毙。
- **瞬间痊愈**：在营地可消耗 50 金币瞬间回满血蓝。

### 数值公式

- **血量成长**（分段曲线）：
  - 1-50 级：`150 + 等级 × 175`
  - 51-100 级：`前段累计 + (等级 - 50) × 320`
  - 100 级以上：`前段累计 + (等级 - 100) × 5400`
- **物理减伤**：`防御力 / (防御力 + 等级 × 40)`，上限 85%。
- **共鸣减伤**：`共鸣 × 0.0015`。
- **总减伤**：`min(95%, 物理减伤 + 共鸣减伤)`。
- **公共冷却（GCD）**：`2500 / (1 + 冷却缩减 / 100)` 毫秒。
- **暴击伤害**：默认 150%。
- **经验需求**（分段指数）：
  - 1-59 级：`100 × 1.15^(等级 - 1)`
  - 60-99 级：`100 × 100 × 1.20^(等级 - 60)`
  - 100 级以上：`100 × 5000 × 1.15^(等级 - 100)`

## 开发者与GM指令

项目原生支持调试指令（在浏览器按 F12 打开控制台使用）：

- 输入 `gm('密码')` 解锁 GM 面板。
- 输入 `gm(1)` / `gm(2)` 等执行对应 GM 功能：
  - `gm(1)`：等级 +100
  - `gm(2)`：增加 1000 万金币
  - `gm(3)`：切换无敌模式
  - `gm(4)`：强制解锁全技能
  - `gm(5)`：生成全套终焉装备
  - `gm(6)`：清除所有存档
- 具体密码和实现请参考源码 `main.js` 中的 `GMTools`。

## 项目结构

- **index.html**：主界面结构与样式
- **js/**：JavaScript 源代码目录
  - **data.js**：技能、装备、宝珠等静态数据库
  - **engine.js**：核心战斗逻辑、数值公式、掉落算法、事件总线
  - **ui.js**：界面渲染与交互控制
  - **main.js**：入口文件、存档管理、GM 工具
- **README.md**：项目说明文档
