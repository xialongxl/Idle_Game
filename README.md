# 末光咏叹 / Idle RPG

> 一个纯文字、高自由度的放置挂机爬塔游戏。无需操作，策略至上。

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

## 核心特色

- **纯文字挂机**：没有任何图片或动画，只有纯粹的数字和策略。战斗实时推进，支持1倍至10倍速调节。
- **深度技能序列**：14个技能槽位自由编排，支持GCD与oGCD机制。可自定义起手爆发与常规循环，实现无缝技能链。
- **高自由度装备系统**：装备动态生成，拥有破损至终焉九大品质。武器加攻击、防具加防御的隔离机制。
- **宝珠镶嵌系统**：通过镶嵌生命、攻击、暴击、共鸣等宝珠，自由调整属性，打造属于自己的流派。
- **营地与风险机制**：每5层设安全区，战败退回。自由设置自动熔炼阈值与安全血线。

## 快速开始

1. 克隆仓库
   git clone https://github.com/xialongxl/Idle_Game.git
2. 打开游戏
   直接双击 index.html 文件即可运行，无需安装任何依赖。
3. 开始爬塔
   点击探索出发按钮即可开始挂机，你可以在右侧面板编排技能序列。

## 游戏核心机制

### 战斗与技能
- GCD与oGCD：大部分技能受公共冷却制约，oGCD技能可无视GCD瞬间释放。
- 优先级循环：战斗严格按照你设定的技能顺序尝试释放，CD或蓝耗不足时自动尝试下一个。
- 持续效果：支持DoT、HoT、护盾、易伤、Buff等多种效果叠加，易伤为乘算。

### 装备与成长
- 动态生成：所有装备均由算法实时生成，楼层越高，高品质爆率越高。
- 词条隔离：武器只带攻击词条，防具只带防御词条，首饰可带攻防及稀有副属性。
- 强化与宝珠：装备可消耗金币强化，击败Boss可获得宝珠并镶嵌提升属性。

### 营地与策略
- 安全层：每5层为一个安全区，战败或手动撤退将退回上一个安全区。
- 自动熔炼：可设置品质阈值，低于阈值的装备掉落时自动分解为金币。

## 开发者与GM指令

在浏览器按F12打开控制台使用：
- 输入 gm('密码') 解锁GM面板。
- 输入 gm(1) 或 gm(2) 等执行对应GM功能。
- 具体密码和功能请参考源码 main.js 中的 GMTools 实现。
